# `IL1LineaTokenBurner`

### NoTokensToBurn

```solidity
error NoTokensToBurn()
```

Error thrown when there are no tokens to burn.

### L1LineaTokenBurnerInitialized

```solidity
event L1LineaTokenBurnerInitialized(address messageService, address lineaToken)
```

Emitted when the L1 Linea Token Burner is initialized.

#### Parameters

| Name | Type | Description |
| ---- | ---- | ----------- |
| messageService | address | The address of the Message Service contract. |
| lineaToken | address | The address of the LINEA token contract. |

### claimMessageWithProof

```solidity
function claimMessageWithProof(struct IL1MessageService.ClaimMessageWithProofParams _params) external
```

Claims a message with proof and burns the LINEA tokens held by this contract.

#### Parameters

| Name | Type | Description |
| ---- | ---- | ----------- |
| _params | struct IL1MessageService.ClaimMessageWithProofParams | The parameters required to claim the message with proof. |
