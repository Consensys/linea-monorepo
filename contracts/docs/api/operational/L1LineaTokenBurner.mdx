### claimMessageWithProof

```solidity

```

Claims a message with proof and burns the LINEA tokens held by this contract.

_This is expected to be permissionless, allowing anyone to trigger the burn._

#### Parameters

| Name | Type | Description |
| ---- | ---- | ----------- |
| _params | struct IL1MessageService.ClaimMessageWithProofParams | The parameters required to claim the message with proof. |
