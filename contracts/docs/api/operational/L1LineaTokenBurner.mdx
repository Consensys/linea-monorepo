# `IL1LineaToken`

### syncTotalSupplyToL2

```solidity
function syncTotalSupplyToL2() external
```

Synchronizes the total supply of the L1 token to the L2 token.

_This function sends a message to the L2 token contract to sync the total supply.
NB: This function is permissionless on purpose, allowing anyone to trigger the sync.
This function can only be called after the L2 token address has been set._
