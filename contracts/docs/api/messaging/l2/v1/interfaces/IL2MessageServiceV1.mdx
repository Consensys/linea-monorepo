# `IL2MessageServiceV1`

### CONTRACT_VERSION

```solidity
function CONTRACT_VERSION() external view returns (string contractVersion)
```

Returns the ABI version and not the reinitialize version.

#### Return Values

| Name | Type | Description |
| ---- | ---- | ----------- |
| contractVersion | string | The contract ABI version. |

### setMinimumFee

```solidity
function setMinimumFee(uint256 _feeInWei) external
```

The Fee Manager sets a minimum fee to address DOS protection.

_MINIMUM_FEE_SETTER_ROLE is required to set the minimum fee._

#### Parameters

| Name | Type | Description |
| ---- | ---- | ----------- |
| _feeInWei | uint256 | New minimum fee in Wei. |
